# DevLab Dashboard

모든 서비스에 빠르게 접속할 수 있는 웹 기반 대시보드입니다.

## 실행 방법

```bash
cd dashboard
npm install
npm run dev
```

브라우저에서 http://localhost:5173 접속

## 접근 가능한 서비스

- ElasticSearch (9200)
- Kibana (5601)
- Redis (6379)
- RedisInsight (5540)
- Kafka UI (8090)
- Prometheus (9090)
- Grafana (3000)
- Spring Boot Actuator (8080)

각 서비스 카드를 클릭하면 새 탭에서 열립니다.
